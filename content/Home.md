# MarkdownWeb

Willkommen bei **MarkdownWeb** – einer schlanken, datenschutzkonformen Dokumentationsplattform, die vollständig auf PHP basiert und auf jedem Standard-Webhosting läuft.

## Was ist MarkdownWeb?

MarkdownWeb wandelt Markdown-Dateien automatisch in eine strukturierte Website um. Die Ordnerstruktur deiner Dateien wird direkt zur Navigation.

## Kernfunktionen

- **Automatische Navigation** aus der Ordnerstruktur
- **Volltext-Suche** mit SQLite FTS5 – skaliert auf beliebig viele Seiten
- **Dark / Light Mode** – gespeichert im Browser
- **Inhaltsverzeichnis** – automatisch aus Überschriften generiert
- **Backlinks** – zeigt welche Seiten auf die aktuelle Seite verlinken
- **Admin-Backend** – Dateien hochladen, verwalten, Suchindex aufbauen
- **DSGVO-konform** – keine externen Ressourcen, kein Tracking

## Voraussetzungen für den Betrieb

| Anforderung | Details |
|---|---|
| PHP | 8.1 oder neuer |
| Erweiterungen | `pdo_sqlite`, `zip`, `mbstring` |
| Webserver | Apache mit `mod_rewrite` |
| Hosting | IONOS, Strato, All-Inkl. und andere ✓ |
